# 📚 BookBrief
> Read Less. Know More.

## 🌐 Live Demo
BookBrief Live

## 📖 About
BookBrief is a curated book summary web app where you can find detailed summaries, key lessons, and ratings of popular books — all in one place.

## 📚 Books Available
- The Silent Patient
- Can We Be Strangers Again?
- Atomic Habits
- The Alchemist
- It Ends With Us
- Gone Girl
- The Da Vinci Code
- The Girl with the Dragon Tattoo
- The Housemaid

## ✨ Features
- 📖 Detailed book summaries
- ⭐ Ratings and genre tags
- 🌙 Dark mode
- 📱 Mobile responsive
- 💬 Book suggestion via WhatsApp
- 🔖 Genre filter menu

## 🛠️ Built With
- HTML
- CSS
- JavaScript
- Netlify (Hosting)
